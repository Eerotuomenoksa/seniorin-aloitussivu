Tee tästä aloitussivu

Näin lisäät tämän sivun aloitussivuksi selaimessa:

Chrome (Windows / Mac)

  1. Avaa kolme pistettä oikeasta yläkulmasta
  2. Valitse Asetukset
  3. Valitse Käynnistyksen yhteydessä
  4. Valitse Avaa tietty sivu tai sivujoukko ja lisää tämä osoite

Firefox

  1. Avaa valikko oikeasta yläkulmasta
  2. Valitse Asetukset
  3. Valitse Käynnistys
  4. Lisää tämän sivun osoite kotisivuksi

Microsoft Edge

  1. Avaa kolme pistettä oikeasta yläkulmasta
  2. Valitse Asetukset
  3. Valitse Käynnistyksen yhteydessä
  4. Lisää tämä sivu avautuvaksi käynnistyksessä

Safari (Mac)

  1. Avaa Safari-valikko ylhäältä
  2. Valitse Asetukset
  3. Kohdasta Yleiset valitse aloitussivuksi tämä osoite
